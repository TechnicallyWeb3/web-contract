Welcome to Web4

What is WTTP?

The Web3 Transfer Transfer Protocol (WTTP) is a revolutionary protocol designed to facilitate seamless linking and redirecting to resources across various blockchains and decentralized storage systems. It introduces custom MIME types and redirect mechanisms, making content retrieval and management in Web3 environments more efficient and user-friendly.

How Does the Browser Work?

The browser is designed to interpret WTTP types, allowing it to handle contract-wide redirects and resource-specific links. It checks the contract's redirect information and follows it according to the specified type and code, ensuring a smooth user experience when navigating decentralized resources.

What is a WebContract?

The WebContract is a smart contract that manages web resources, admin roles, and various token operations. It allows for the storage, management, and transfer of files and tokens, including Ether, ERC20, and ERC721 tokens. The WebContract supports ERC4804 URL standards, ensuring interoperability with other decentralized applications.

Why Web4?

Web4 represents the next evolution of the internet, where decentralized technologies empower users with more control over their data and interactions. By leveraging WTTP and WebContracts, Web4 aims to create a more open, secure, and user-centric web experience.
